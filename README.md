# COSMOS Environmental Sensor Node

Open Arduino-based environmental sensing module for COSMOS nodes.

## Project Overview

This project develops a compact environmental sensing package for deployment on COSMOS infrastructure. The node collects hyperlocal atmospheric and environmental data using multiple integrated sensors.

## Current Sensors

- AmbiMate MS4
  - Temperature
  - Humidity
  - Light
  - TVOC
  - eCO2
  - Motion

- BMP085
  - Pressure
  - Altitude
  - Temperature

- BOB-09868 Microphone
  - Peak-to-Peak Acoustic Measurements

- SPS30 Particle Sensor
  - PM1.0
  - PM2.5
  - PM4.0
  - PM10

## Hardware

- Arduino Uno
- Breadboard Prototype
- KiCad Schematics

## Status

Currently integrating sensors and validating measurements prior to PCB design and deployment.
